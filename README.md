# 💼 Salary Prediction App using Machine Learning

This project is a **Machine Learning Web Application** that predicts salary based on a person's **Years of Experience** using **Simple Linear Regression**.  
The model is trained using Python and deployed with **Streamlit** for an interactive user experience.

---

## 🚀 Project Overview

The main goal of this project is to understand and implement the complete Machine Learning workflow:

- Data Loading & Preprocessing  
- Model Training using Linear Regression  
- Data Visualization  
- Model Evaluation  
- Saving Model using Pickle  
- Deployment using Streamlit  

Users can enter their years of experience, and the app predicts the estimated salary instantly.

---

## 📊 Dataset

The dataset contains two columns:

- **YearsExperience** → Number of years worked  
- **Salary** → Salary earned  

This dataset is commonly used for beginners to learn regression models.

## 📈 Machine Learning Algorithm

### Simple Linear Regression

Used to predict salary based on one independent variable:

- Input → Years of Experience  
- Output → Predicted Salary  

Equation:

Salary = m(Experience) + c

Where:

- **m** = slope  
- **c** = intercept
